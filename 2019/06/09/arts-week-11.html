    Algorithm

    5087. 活字印刷

    你有一套活字字模 tiles，其中每个字模上都刻有一个字母 tiles[i]。返回你可以印出的非空字母序列的数目。

        示例 1：
    
    输入："AAB"
    输出：8
    解释：可能的序列为 "A", "B", "AA", "AB", "BA", "AAB", "ABA", "BAA"。
    示例 2：
    
    输入："AAABBC"
    输出：188

    提示：
    
    1 <= tiles.length <= 7
    tiles 由大写英文字母组成

    解题思路：

    这道题是一道典型的回溯算法，可以用递归解法。

    解题代码：

        class Solution:
        def numTilePossibilities(self, tiles: str) -> int:
            from collections import Counter
            from copy import copy
            counter = dict(Counter(tiles))
            ans = []
            def _helper(co, s):
                if not co:
                    return 
                
                for c in co.keys():
                    _co = copy(co)
                    _co[c] -= 1
                    ans.append(s + c)
                    if _co[c] == 0:
                        _co.pop(c)
                    _helper(_co, s + c)
            
            _helper(counter, '')
            return len(ans)
